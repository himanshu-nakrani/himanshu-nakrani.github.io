# Product

This is a personal portfolio website for Himanshu Nakrani, a Generative AI / Production AI Engineer. The site showcases his professional experience, projects, skills, research publications, and profiles (GitHub, Kaggle, LeetCode).

## Key Sections
- Hero / About — intro, stats, social links
- Experience — work history at State Street Corporation and others
- Projects — AI/ML projects (Alpha Copilot, Agent Forge, WealthAI, RAG systems, etc.)
- Skills — categorized tech skills
- Research — IEEE publications on Text-to-SQL and Graph-of-Thoughts
- Profiles — GitHub contributions, Kaggle notebooks/datasets, LeetCode stats
- Live Metrics Dashboard — real-time activity aggregation
- Contact — footer contact section

## Responsive Strategy
- Desktop: multi-page SPA with React Router (separate routes per section)
- Mobile: single all-in-one scrollable page (`MobileAllInOnePage`) — sub-pages redirect to `/` on mobile
